JavaScript: Logika

Porównania

Porównania zapisuje w następujący sposób:

Porównania wykorzystujemy w słowie kluczowym if (jeżeli):

if ('JavaScript' != 'Java'){
// wykonaj kod jeśli true
console.log('Java to nie JavaScript');
}
Operatory porównania
  • > - lewa strona większa od prawej
  • <  - lewa strona mniejsza od prawej
  • >=  - lewa strona większa / równa od prawej
  • <= - lewa strona mniejsza / równa od prawej
  • ==  - obie strony równe
  • ===  - obie strony równe - porównanie ścisłe (sprawdza typeof)
  • !=  - obie strony różne
  • !==  - obie strony różne (sprawdza typeof

W JavaScript'cie istnieją dwa sposoby porównywania czy dane wartości są takie same.

Pierwsza metoda to "==",  która nie sprawdza typów zmiennych tylko porównuje ich wartości, konwertuje je (czyli 2 =='2' => prawda)

Druga metoda to porównanie ścisłe '===', czyli oprócz porównania wartości to porównuje typy zmiennych (czyli 2 ===  '2' => false)

typeof zmienna pokazuje typ zmiennej

Wartości fałszywe (Falsy)

Wyróżniamy 6 wartości, które zawsze zwracają nam fałsz:

  • false
  • 0
  • '', "", `` - pusty string
  • null
  • undefined
  • NaN - not a number - wyskakuje nam, kiedy próbujemy robić rzeczy związane z liczbami na zmiennych nie będącymi liczbami
Wartości prawdziwe (Truthy)

To cała reszta wartości. Czyli:

  • 'Tekst'
  • 12
  • -4
  • {}
  • []
  • true
  • itd.
Warunki logiczne
  • AND  && - wszystkie warunki muszą być true if ((myVariable == 4) && (otherVar !== 'test')) {...}
  • OR  || - przynajmniej jeden warunek ma być true if ((myVariable == 4) || (otherVar !== 'test')) {...}
  • NOT ! - odwraca wartość wyrażenia logicznego if (!('JavaScript' == 'Java')) {
    - // ten kod się wykona
    - }
    !false => true
    !true => false
Słowa kluczowe
  • if - - pozwala na budowanie bloku kodu, który się wykona jeśli warunek if'a będzie prawdą if ('JavaScript' != 'Java'){
    // wykonaj kod jeśli true
    console.log('Java to nie JavaScript');
    }
  • else - pozwala na budowanie bloku kodu, który się wykona jeśli warunek if'a będzie fałszem if (myVariable == 4) {
    // wykonaj kod jeśli true
    } else {
    // wykonaj kod jeśli false
    }
  • else if  - pozwala na budowanie bloku kodu, który się wykona jeśli warunek if'a będzie fałszem i warunek zdefiniowany przy else if  będzie prawdą if (myVariable == 4) { // wykonaj kod jeśli true } else if (myVariable == 8) { // wykonaj kod jeśli myVariable == 8 }
  • switch - pozwala na budowanie bloku kodu, w których definiujemy case (przypadek) w którym dany kod ma się wykonać, oraz wartość domyślną, która ma przypisany blok kodu, który się wykona jeśli przypadki wyżej nie są spełnione switch (myVariable) { case 4: console.log('hi 4'); break; case 8: console.log('hi 8'); break; default: console.log('default response'); }
  • zapis skrócony - po pytajniku jest if, a po dwukropku else (zmienna1 > zmienna2) ? console.log('true') : console.log('false')
  • Wynik operacji logicznych możemy zwracać do zmiennej (obowiązuje to tylko obiekty proste np. boolean, liczby oraz obiekty) const result =
    (32 > 20) ? true : false;

    console.log(result); // to prawda
